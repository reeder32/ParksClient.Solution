# Get Started

- Open your favorite Terminal

## Installation

```sh
git clone https://github.com/reeder32/ParksClient.Solution.git
```

##### Open in VS Code

```sh
cd ParksClient.Solution
code .
```

#### Start Server

```sh
cd Parks
dotnet run
```

#### Run on Browser

```sh
cd PierresBakery
dotnet run
```

[Open your browser here](http://localhost:4000)
